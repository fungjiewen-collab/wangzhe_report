# 🚀 5分钟快速开始

## ✅ 前置要求

- 已安装 Git
- 拥有 GitHub 账号
- 已安装 Python 3.6+

---

## 📝 操作步骤

### Step 1: 初始化仓库（1分钟）

```bash
cd wangzhe_report
./init_github.sh
```

按提示输入你的GitHub用户名和邮箱。

### Step 2: 创建GitHub仓库（1分钟）

1. 访问 [https://github.com/new](https://github.com/new)
2. 仓库名：`wangzhe_report`
3. 设为 **Public**
4. **不要**勾选 "Initialize with README"
5. 点击 "Create repository"

### Step 3: 推送代码（1分钟）

```bash
# 替换 YOUR_USERNAME 为你的GitHub用户名
git remote add origin https://github.com/YOUR_USERNAME/wangzhe_report.git
git branch -M main
git push -u origin main
```

### Step 4: 启用GitHub Pages（1分钟）

1. 进入仓库 → **Settings** → **Pages**
2. Source: Branch=`main`, Folder=`/(root)`
3. 点击 **Save**
4. 等待1-2分钟

### Step 5: 访问报告（1分钟）

访问网址：
```
https://YOUR_USERNAME.github.io/wangzhe_report/
```

🎉 完成！你现在拥有了固定网址的在线报告！
